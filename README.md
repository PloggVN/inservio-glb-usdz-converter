# GLB to USDZ Converter

A simple and efficient tool to convert 3D models from **GLB/GLTF** format to **USDZ** directly in your browser. This tool is built using **React** and **Three.js**, ensuring that your models are processed locally without ever leaving your machine.

## How to Run

Follow these steps to get the project running locally:

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start the development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

Once the server is running, simply open the provided URL in your browser, upload your `.glb` or `.gltf` file, and download the converted `.usdz` file instantly.
